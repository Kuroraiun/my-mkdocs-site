# モバイルICOCAの概要

ICOCAの公式アプリ。主なポイントは以下のとおり。

| 項目 | 内容 |
|------|------|
| ICOCAの種類 | 定期券の有無を選択できる。 |
| ICOCAの名前 | 複数枚のICOCAを発行できる。<br>それぞれに「ニックネーム（カード名）」を設定できる（例：プライベート用、仕事用など）。|
| メインカードの設定 | 交通系ICカードを1枚、メインカードとして設定できる。<br>ICOCA以外のカード（Suicaなど）も設定可能。<br>設定方法は以下の公式サイトのとおり。<br>https://www.jr-odekake.net/icoca/mobileicoca/start/main/  |
| WESTERポイント | モバイルICOCAアプリでJR西日本の電車に乗ると、WESTERポイントが貯まる。<br>WESTERポイントには有効期限がある。<br>WESTERポイントはICOCAへのチャージなどに利用できる。 |
