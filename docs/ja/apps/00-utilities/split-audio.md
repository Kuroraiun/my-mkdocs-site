# 音源を分離する方法（Go-Splitter）

DAW向けプラグインなどを開発する **SoliderSound Ltd.** が、AI音源分離ツール **「Go-Splitter」** の Windows対応版をリリース。
音楽ファイルをAIでパートごとに分離できる無料ソフト。

## 主な特徴

* **対応OS**：Windows 10 / Windows 11
* **対応ファイル形式**：WAVE、MP3、FLAC、M4A、AAC、AIFF、Ogg Vorbis など
* **分離可能パート**：

  * ボーカル
  * ベース
  * ドラム
  * その他
    → 複数トラックへの分離が可能。

## 分離後の出力

* 各パートは **WAVE形式で出力**。
* ZIP形式でまとめて保存可能。
* **ローカル処理** のため、オフライン環境でも使用可能。

## 使い勝手・品質

* 分離クオリティは高め。
* やや「シュワシュワ」感が残る場合あり。
* エフェクト処理で実用レベルまで調整可能。
* 品質は **3段階設定** で切り替え可能。
