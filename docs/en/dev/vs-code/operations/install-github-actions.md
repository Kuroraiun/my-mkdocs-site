# Install the GitHub Actions extension

1. Click **Install**.

    ![Screenshot](install-github-actions-01.webp)

1. Click **Sign in to GitHub**.

    ![Screenshot](install-github-actions-02.webp)

1. Click **Allow**.

    ![Screenshot](install-github-actions-03.webp)

1. Click **Continue**.

    ![Screenshot](install-github-actions-04.webp)

1. Click **Open Visual Studio Code**.

    ![Screenshot](install-github-actions-05.webp)

1. The installation is complete.

    ![Screenshot](install-github-actions-06.webp)
