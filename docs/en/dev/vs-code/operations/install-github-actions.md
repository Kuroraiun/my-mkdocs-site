# Install the GitHub Actions extension

1. Click **Install**.

    ![Screenshot](install-github-actions-01.png)

1. Click **Sign in to GitHub**.

    ![Screenshot](install-github-actions-02.png)

1. Click **Allow**.

    ![Screenshot](install-github-actions-03.png)

1. Click **Continue**.

    ![Screenshot](install-github-actions-04.png)

1. Click **Open Visual Studio Code**.

    ![Screenshot](install-github-actions-05.png)

1. The installation is complete.

    ![Screenshot](install-github-actions-06.png)
