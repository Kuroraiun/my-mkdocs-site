# Use the nano editor

1. Open the Ubuntu terminal.
   ![Screenshot](use-nano-editor-01.webp)

2. Create a new file and open it:

      ```bash
      nano Dockerfile
      ```
   ![Screenshot](use-nano-editor-02.webp)

4. Write your content, then press **Ctrl + O** to save the file.
   ![Screenshot](use-nano-editor-03.webp)

5. Press **Enter** to confirm the file name.
   ![Screenshot](use-nano-editor-04.webp)

6. Press **Ctrl + X** to exit.
   ![Screenshot](use-nano-editor-05.webp)
