# Cancel a ChatGPT subscription

1. Select **Settings**.
    ![Screenshot](cancel-chatgpt-01.webp)

1. Select **Account**.
    ![Screenshot](cancel-chatgpt-02.webp)

1. Select **Manage** → **Cancel Subscription**.
    ![Screenshot](cancel-chatgpt-03.webp)

1. Click **Cancel subscription**.
    ![Screenshot](cancel-chatgpt-04.webp)

1. Confirm the expiration date.
    ![Screenshot](cancel-chatgpt-05.webp)
