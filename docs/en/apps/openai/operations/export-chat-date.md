# Export chat data from ChatGPT

Export chat history to download a local copy of your conversations and account data.

## Steps

1. Open ChatGPT.

2. Click your **profile icon** in the bottom-left corner.

3. Select **Settings**.

4. Open **Data Controls**.

5. Click **Export Data**.

6. Click **Confirm export**.

7. Check your email for the export message.

8. Download the attached ZIP file from the email link.

## Note

The downloaded ZIP file typically contains:

* `chat.html` — human-readable version of conversations
* `conversations.json` — structured conversation data
* Additional JSON files with account and activity data

Open `chat.html` in a web browser to read the exported conversations. Use `conversations.json` for data processing or migration to other tools.
