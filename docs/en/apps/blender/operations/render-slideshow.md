# Render a slideshow

1. Open **Video Editing**.
   ![Screenshot](render-slideshow-01.webp)

2. Select the scene.
   ![Screenshot](render-slideshow-02.webp)

    !!! Tip  
         Select the same scene in both the timeline and the Properties panel.

3. Open **Format**.
   ![Screenshot](render-slideshow-03.webp)

4. Select **1920 × 1080**.
   ![Screenshot](render-slideshow-04.webp)

    !!! Note  
         **1920 × 1080** means Full HD.

5. Select **30 fps** for the slideshow.
   ![Screenshot](render-slideshow-05.webp)

6. Set the frame range by selecting the start frame and end frame.
   ![Screenshot](render-slideshow-06.webp)

7. Select the output folder.
   ![Screenshot](render-slideshow-07.webp)

8. Select **Video**, then choose the color mode.
   ![Screenshot](render-slideshow-08.webp)

    !!! Note  
         **BW** means **black and white**.

9. Select **MPEG-4**.
   ![Screenshot](render-slideshow-09.webp)

10. Select **H.264**.
    ![Screenshot](render-slideshow-10.webp)

    !!! Note  
         **H.264** produces an **MP4 video**.


11. Select **Render Animation**.
    ![Screenshot](render-slideshow-11.webp)

12. The MP4 video is generated.
    ![Screenshot](render-slideshow-12.webp)
