# Run Google security checkup

Use Google Security Checkup to review and improve the security of a Google account. The process verifies login activity, recovery settings, devices, and connected apps.

## Steps

1. Open the Security Checkup page.

    https://myaccount.google.com/security-checkup

2. Sign in with the Google account you want to review.

3. Review **Recent security activity**.

    * Check for unknown login locations.
    * Check for unfamiliar devices.
    * If an activity is suspicious, select **This wasn't me** and follow the prompts.

4. Verify **Sign-in & recovery** settings.

    * Confirm the password is secure.
    * Enable **2-Step Verification**.

    To enable it:

    ```
    1. Open **2-Step Verification**.
    2. Select a verification method:
        - SMS code
        - Authenticator app
        - Passkey
    ```

5. Confirm **Recovery information**.

    * Verify the recovery email address.
    * Verify the recovery phone number.
    * Replace outdated contact information.

6. Review **Your devices**.

    * Check the list of signed-in devices.
    * Select **Sign out** for devices that are not recognized.

7. Review **Third-party access**.

    * Check apps connected through **Sign in with Google**.
    * Remove applications that are unused or untrusted.

## Note

Run Security Checkup periodically to maintain account security. Review settings again after:

* Suspicious login alerts
* Password changes
* Installing new browser extensions
* Connecting new apps to the account
