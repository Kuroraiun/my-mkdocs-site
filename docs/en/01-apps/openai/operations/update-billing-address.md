# Update a ChatGPT billing address

1. Click the profile icon.
   ![Screenshot](00-images/update-billing-address01.png)

2. Select **Settings**.
   ![Screenshot](00-images/update-billing-address02.png)

3. Select **Account**.
   ![Screenshot](00-images/update-billing-address03.png)

4. Click **Manage** under **Payment**.
   ![Screenshot](00-images/update-billing-address04.png)

5. Click **Update information**.
   ![Screenshot](00-images/update-billing-address05.png)

6. Update the billing information.
   ![Screenshot](00-images/update-billing-address06.png)
