# Update a ChatGPT billing address

1. Click the profile icon.
   ![Screenshot](update-billing-address-01.png)

2. Select **Settings**.
   ![Screenshot](update-billing-address-02.png)

3. Select **Account**.
   ![Screenshot](update-billing-address-03.png)

4. Click **Manage** under **Payment**.
   ![Screenshot](update-billing-address-04.png)

5. Click **Update information**.
   ![Screenshot](update-billing-address-05.png)

6. Update the billing information.
   ![Screenshot](update-billing-address-06.png)
