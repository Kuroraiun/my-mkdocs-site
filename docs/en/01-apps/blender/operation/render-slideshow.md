# Render a slideshow

1. Open **Video Editing**.
   ![Screenshot](../00-images/render-slideshow01.png)

2. Select the scene.
   ![Screenshot](../00-images/render-slideshow02.png)

    !!! Tip  
         Select the same scene in both the timeline and the Properties panel.

3. Open **Format**.
   ![Screenshot](../00-images/render-slideshow03.png)

4. Select **1920 × 1080**.
   ![Screenshot](../00-images/render-slideshow04.png)

    !!! Note  
         **1920 × 1080** means Full HD.

5. Select **30 fps** for the slideshow.
   ![Screenshot](../00-images/render-slideshow05.png)

6. Set the frame range by selecting the start frame and end frame.
   ![Screenshot](../00-images/render-slideshow06.png)

7. Select the output folder.
   ![Screenshot](../00-images/render-slideshow07.png)

8. Select **Video**, then choose the color mode.
   ![Screenshot](../00-images/render-slideshow08.png)

    !!! Note  
         **BW** means **black and white**.

9. Select **MPEG-4**.
   ![Screenshot](../00-images/render-slideshow09.png)

10. Select **H.264**.
    ![Screenshot](../00-images/render-slideshow10.png)

    !!! Note  
         **H.264** produces an **MP4 video**.


11. Select **Render Animation**.
    ![Screenshot](../00-images/render-slideshow11.png)

12. The MP4 video is generated.
    ![Screenshot](../00-images/render-slideshow12.png)
