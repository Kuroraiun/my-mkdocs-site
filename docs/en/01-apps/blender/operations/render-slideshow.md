# Render a slideshow

1. Open **Video Editing**.
   ![Screenshot](render-slideshow-01.png)

2. Select the scene.
   ![Screenshot](render-slideshow-02.png)

    !!! Tip  
         Select the same scene in both the timeline and the Properties panel.

3. Open **Format**.
   ![Screenshot](render-slideshow-03.png)

4. Select **1920 × 1080**.
   ![Screenshot](render-slideshow-04.png)

    !!! Note  
         **1920 × 1080** means Full HD.

5. Select **30 fps** for the slideshow.
   ![Screenshot](render-slideshow-05.png)

6. Set the frame range by selecting the start frame and end frame.
   ![Screenshot](render-slideshow-06.png)

7. Select the output folder.
   ![Screenshot](render-slideshow-07.png)

8. Select **Video**, then choose the color mode.
   ![Screenshot](render-slideshow-08.png)

    !!! Note  
         **BW** means **black and white**.

9. Select **MPEG-4**.
   ![Screenshot](render-slideshow-09.png)

10. Select **H.264**.
    ![Screenshot](render-slideshow-10.png)

    !!! Note  
         **H.264** produces an **MP4 video**.


11. Select **Render Animation**.
    ![Screenshot](render-slideshow-11.png)

12. The MP4 video is generated.
    ![Screenshot](render-slideshow-12.png)
