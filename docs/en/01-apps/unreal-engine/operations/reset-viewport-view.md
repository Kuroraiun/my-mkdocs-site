# Reset Viewport View

Use these methods to restore the Viewport to a normal view.

## Return to Perspective view

1. Click the view mode selector in the top-left corner of the Viewport.
2. Select **Perspective**.

This restores the standard 3D view.

---

## Reposition the camera

1. Hold the **right mouse button**.
2. Press **W, A, S, D, Q, or E** to move the camera.
3. Move the mouse to adjust the viewing angle.

This lets you manually return to the desired view.

---

## Restore focus to an Actor (optional)

1. Select an Actor.
2. Press **F**.

The Viewport centers on the selected Actor.

Note: This focuses the camera on an Actor. It does not restore a default Viewport position.

---

## Restore default editor layout (if Viewport is missing)

1. Click **Window** → **Load Layout** → **Default Editor Layout**.

This restores the default Viewport and panel layout.
