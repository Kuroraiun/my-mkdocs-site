# Change folder access role in Google Drive

Change the access role of a user for a shared folder.

## Steps

1. Open **Google Drive**.
2. Right-click the folder.
3. Click **Share**.
4. In the access list, locate the user.
5. Click the role dropdown next to the user name.
6. Select one of the following:

   * **Viewer**
   * **Commenter**
   * **Editor**

7. Click **Done**.

## Optional

### Role descriptions

* **Viewer**: Can view files only.
* **Commenter**: Can view and add comments.
* **Editor**: Can view, comment, edit, add, and delete files.

### Remove access

1. Open **Share**.
2. Click the dropdown next to the user.
3. Select **Remove access**.
4. Click **Done**.
