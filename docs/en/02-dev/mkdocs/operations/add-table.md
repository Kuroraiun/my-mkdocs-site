# Add a table 

## Align a column 
Use colons in the separator row:
```
| English | Japanese |
|:-------:|:--------:|
| You're not imagining it. | 君の気のせいじゃないよ。 |
```
The table will look something like this:

| English | Japanese |
|:-------:|:--------:|
| You're not imagining it. | 君の気のせいじゃないよ。 |

> Notes:

* `:---` - Left aligned
* `---:` - Right aligned
* `:---:` - Centered

## Escape the | (pipe) character 
Use a backslash (or a yen mark):
```bash title="terminal"
\|
```
