
# Add a copyright footer in MkDocs Material
Add the copyright text to the YAML file:
```yaml title="mkdocs.yml"
site_name: MyWebsite

copyright: © 2026 MyWebsite. All rights reserved.

theme:
  name: material
```

The footer will look something like this:
```bash title="MkDocs"
© 2026 MyWebsite. All rights reserved.
Made with Material for MkDocs
```

!!! Note
    Do not remove the following default text.
    ```bash title="MkDocs"
    Made with Material for MkDocs
    ```
    It’s not mandatory but it is polite to keep it.
