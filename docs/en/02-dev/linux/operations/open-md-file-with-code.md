# Open a Markdown file using the code command

Open a Markdown file in Visual Studio Code using its full file path.

## Steps

1. Open a terminal.

2. Run:

    ```
    code /full/path/to/filename.md
    ```

    Example:

    ```
    code /Users/username/docs/notes/example.md
    ```

    The file opens in Visual Studio Code.

## Note

* On Windows, use:

  ```
  code C:\Users\username\docs\notes\example.md
  ```

* If the path contains spaces, wrap it in quotes:

  ```
  code "/full/path/with spaces/example.md"
  ```
