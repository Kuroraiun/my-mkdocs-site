# Install the GitHub Actions extension

1. Click **Install**.

    ![Screenshot](00-images/install-github-actions01.png)

1. Click **Sign in to GitHub**.

    ![Screenshot](00-images/install-github-actions02.png)

1. Click **Allow**.

    ![Screenshot](00-images/install-github-actions03.png)

1. Click **Continue**.

    ![Screenshot](00-images/install-github-actions04.png)

1. Click **Open Visual Studio Code**.

    ![Screenshot](00-images/install-github-actions05.png)

1. The installation is complete.

    ![Screenshot](00-images/install-github-actions06.png)
