# Cancel or clear commands

Use keyboard shortcuts to cancel, edit, or clear commands in the terminal.

## Steps

### Cancel a command before pressing Enter

* Clear the entire line:

  ```
  Ctrl + U
  ```

* Delete one word:

  ```
  Ctrl + Backspace
  ```

* Move to the beginning of the line:

  ```
  Ctrl + A
  ```

* Move to the end of the line:

  ```
  Ctrl + E
  ```

### Stop a running command

If the command is already running, press:

```
Ctrl + C
```

### Clear the terminal screen

* Run:

  ```
  clear
  ```

* Or press:

  ```
  Ctrl + L
  ```

## Note

These shortcuts work in most shells such as `bash` and `zsh` inside Visual Studio Code.
