# Get started with Bitwarden

Install Bitwarden and configure basic security settings.

## Steps

1. Download and install Bitwarden for your operating system.

    [https://bitwarden.com/download/](https://bitwarden.com/download/)

2. Add the browser extension for your browser.

3. Log in to your account and add passwords to your vault.

    [https://vault.bitwarden.com/#/login?utm_source=welcome_email&utm_medium=email](https://vault.bitwarden.com/#/login?utm_source=welcome_email&utm_medium=email)

4. Enable biometric authentication.

    [https://bitwarden.com/help/biometrics/](https://bitwarden.com/help/biometrics/)
